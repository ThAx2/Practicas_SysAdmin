# Práctica DHCP - Automatización e Idempotencia
**Autor:** Alexander Vega / Ax2  
**Fecha:** 10 de Febrero, 2026

## Descripción
Este repositorio contiene scripts para la configuración automática de servidores DHCP en entornos Linux (Ubuntu) y Windows (Server Core).

## Estructura
- **Linux scripts:** Contiene la lógica en Bash para instalar, configurar y monitorear el servicio `isc-dhcp-server`.
- **Windows script:** Contiene el script de PowerShell para gestionar el rol de DHCP en Windows Server.
